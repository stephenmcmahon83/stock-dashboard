Weekly Stock Market Data Dashboard

Enter a stock ticker (e.g., AAPL, MSFT, GOOG) to view its historical weekly open and close prices.

Analyst's Warning: This demo fetches data directly from an unofficial Yahoo Finance endpoint via a public CORS proxy. This method is inherently unstable and can break without warning if Yahoo changes its API or the proxy goes down. It is not recommended for production use. For reliable data, a server-side solution with an official API provider is the professional standard.
