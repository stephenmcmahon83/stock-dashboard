Advanced Weekly Stock Analysis

Enter a stock ticker (e.g., AAPL, NVDA, SPY) to view its historical weekly performance and statistical summary.

Security Warning: For this demo to work, you must place your API key in the `script.js` file. Be aware that this makes your key visible in the browser. For a real application, API keys should always be kept on a secure backend server, not in client-side code.
